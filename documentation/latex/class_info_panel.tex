\doxysubsection{Detailed Description}
Panel that displays statistical analysis of air quality data. 

The \doxylink{class_info_panel}{Info\+Panel} class is responsible for displaying statistical information about air quality measurements including maximum, minimum, average values, and trend information for selected pollutants.
