\doxysubsection{Detailed Description}
A dialog that displays a list of sensors and allows the user to select one. 

This dialog presents a list of sensors parsed from JSON data and provides functionality for selecting a sensor either by double-\/clicking on it or by selecting it and clicking the OK button.
